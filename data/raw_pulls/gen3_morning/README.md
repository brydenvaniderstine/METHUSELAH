Morning pulls. Contains raw Gen3 BLE pull output files captured at or near waking. Naming convention: gen3_pull_YYYYMMDD_HHMMSS.txt. Note: folder name reflects time-of-day convention only — classifier output (SLEEP WINDOW vs ACTIVE WINDOW) is recorded in the pull file itself and in the comparison log, not in the folder structure. Critical timing constraint: pull before standing up — the 255-event buffer is fully replaced within ~108 seconds of sustained walking. See data/findings/known_issues.md for the confirmed operational pattern.
