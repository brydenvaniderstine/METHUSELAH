Evening pulls. Contains raw Gen3 BLE pull output files from later in the day. Naming convention matches gen3_morning/: gen3_pull_YYYYMMDD_HHMMSS.txt. Note: folder name reflects time-of-day convention only — classifier output (SLEEP WINDOW vs ACTIVE WINDOW) is recorded in the pull file itself and in the comparison log, not in the folder structure. This naming convention is provisional and may be revised.
