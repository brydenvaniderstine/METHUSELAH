# pipeline/data/findings/

Decoder research artifacts — roadmap, known issues, comparison CSVs, baselines.
These are living documents updated as RE work progresses. Not code.

## Files
- `known_issues.md` — per-decoder findings, falsified hypotheses, ceilings, pull logs
- `open_ring_roadmap.md` — full 55-entry decoder status tracker
- `gen3_vs_gen4_comparison.csv` — night-by-night cross-validation of Gen3 decoded values against Gen4 official output
- `gen4_baselines.md` — personal percentile reference bands from 359 nights of Gen4 data
- `ring_decoder_inventory.md` — initial inventory of all open_ring decoders
- `sleep_state_findings.md` — early sleep state decoder notes
